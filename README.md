# Tp-algo
# TP AlgoBox – Introduction à GitHub

## Description du projet

Ce dépôt GitHub contient un TP d’introduction à l’utilisation de **Git** et **GitHub** dans le cadre d’un projet **AlgoBox**.

L’objectif du projet est de prendre en main les commandes Git de base (initialisation d’un dépôt, ajout de fichiers, commit, push, pull) et de comprendre le fonctionnement d’un dépôt distant GitHub.

Les fichiers présents dans ce dépôt sont des algorithmes écrits avec le logiciel **AlgoBox**.

---

## Prérequis

Avant d’ouvrir ou d’exécuter les fichiers du projet, il est nécessaire de disposer de :

* Un ordinateur sous Windows, Linux ou macOS
* Le logiciel **AlgoBox** installé sur la machine
* **Git** installé (facultatif pour l’exécution, nécessaire pour la gestion du dépôt)
* Un compte GitHub (pour la collaboration et la synchronisation)

---

## Contenu du dépôt

Le dépôt est organisé de la manière suivante :

* `tp-algobox/`

  * `exo1.alg` : premier exercice AlgoBox
  * `exo2.alg` : deuxième exercice AlgoBox
  * `exo3.alg` : troisième exercice AlgoBox
* `README.md` : fichier d’explication du projet

Chaque fichier `.alg` correspond à un algorithme indépendant pouvant être ouvert et exécuté séparément.

---

## Comment ouvrir un fichier AlgoBox

1. Lancer le logiciel **AlgoBox**
2. Cliquer sur **Fichier → Ouvrir**
3. Sélectionner le fichier souhaité (`exo1.alg`, `exo2.alg` ou `exo3.alg`)
4. Le code de l’algorithme s’affiche dans l’éditeur AlgoBox

---

## Comment exécuter un algorithme

Une fois le fichier ouvert dans AlgoBox :

1. Vérifier que l’algorithme est correctement chargé
2. Cliquer sur le bouton **Exécuter** (icône ▶)
3. Suivre les instructions affichées (saisies clavier, affichages, résultats)

L’exécution permet de tester le bon fonctionnement de l’algorithme et de vérifier la logique du programme.

